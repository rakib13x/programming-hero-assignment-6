Question No. 1

Question: Discuss the scope of var, let, and const

Answer:

  • var : var is function-scoped and hoisted. It allows redeclaration and reassignment.
  • let : let is block-scoped, hoisted, and allows reassignment but not redeclaration.
  • const : const is block-scoped, hoisted, and does not allow reassignment but allows modification of object properties or array elements.

Question No. 2

Question:Tell us the use cases of null and undefined

Answer:

  • Undefined:
  • Return Value : If a function doesn't explicitly return a value, it implicitly returns undefined.
  • Here's an Example:

    • function noReturn() {

    // This function has no return statement

    }

    const result = noReturn();

    console.log(result); // Outputs: undefined

  • Null:
  • Resetting Values : You can set a variable or object property to null to indicate that it should no longer reference any meaningful value.
  • Here's an Example:

    • let x = null; // x explicitly has no value

    const obj = {

    property: null // Explicitly setting a property to null

    };

Question No. 3

Question:What do you mean by REST API?

Answer:

  • Rest API:

    • REST API, which stands for Representational State Transfer Application Programming Interface, is an architectural style for designing networked applications. It is a set of constraints and principles used to create web services that are scalable, maintainable, and can easily interact with one another over the internet
